# Lista de Tarefas

Uma aplicação moderna e minimalista de lista de tarefas construída com Vue.js.

## 💡 Funcionalidades

- ✅ Adicionar novas tarefas
- ✔️ Marcar tarefas como concluídas
- 🗑️ Remover tarefas
- 📱 Design responsivo
- 🎨 Interface limpa e moderna

## 🛠️ Tecnologias Utilizadas

- Vue.js
- CSS Moderno
- Google Fonts (Inter)
